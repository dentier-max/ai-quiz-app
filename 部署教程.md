# 🍃 AI通识B 刷题助手 — 部署到互联网教程

## 为什么 localhost 关机后不能访问？

`http://localhost:8080` 是你自己电脑上的本地地址，只有你自己的电脑能打开。关机后这个地址就不存在了。

要让别人也能访问，需要把文件放到**公网服务器**上。

---

## 推荐方案：GitHub Pages（完全免费）

**优点：**
- 完全免费，无需购买服务器
- 修改文件后网站自动更新
- 国内/国外都能访问
- 支持自定义域名（可选）

**缺点：**
- 需要注册一个 GitHub 账号（免费）
- 国内访问速度一般（可搭配 CDN 加速）

---

## 部署步骤（5分钟完成）

### 第一步：注册 GitHub 账号

1. 打开 https://github.com/signup
2. 用邮箱注册（推荐用QQ邮箱或学校邮箱）
3. 验证邮箱

### 第二步：创建仓库

1. 登录 GitHub 后，点击右上角 **+** → **New repository**
2. 仓库名称填写：`ai-quiz-app`（或任意名称）
3. 选择 **Public**（公开）
4. 勾选 **Add a README file**
5. 点击 **Create repository**

### 第三步：上传文件

**方法A：网页直接上传（最简单）**

1. 进入刚创建的仓库页面
2. 点击 **Add file** → **Upload files**
3. 把 `ai-quiz-app.html` 文件拖进去
4. 点击 **Commit changes**

**方法B：用 Git 命令（适合后续频繁修改）**

```bash
# 1. 安装 Git（https://git-scm.com/download/win）
# 2. 打开命令行，进入文件所在目录
cd "C:\Users\你的用户名\...\ai-quiz-app.html所在的文件夹"

# 3. 初始化仓库
git init
git add ai-quiz-app.html
git commit -m "first commit"

# 4. 关联远程仓库（把下面的用户名换成你的）
git remote add origin https://github.com/你的用户名/ai-quiz-app.git

# 5. 推送
git branch -M main
git push -u origin main
```

### 第四步：开启 GitHub Pages

1. 在仓库页面，点击 **Settings**（设置）
2. 左侧菜单点击 **Pages**
3. **Source** 选择 **Deploy from a branch**
4. **Branch** 选择 **main** → **/(root)**
5. 点击 **Save**

等待 1-2 分钟，页面会显示你的网址：
```
https://你的用户名.github.io/ai-quiz-app/
```

**注意：** 因为文件名是 `ai-quiz-app.html`，访问时需要加上文件名：
```
https://你的用户名.github.io/ai-quiz-app/ai-quiz-app.html
```

或者把文件名改成 `index.html`，就可以直接访问：
```
https://你的用户名.github.io/ai-quiz-app/
```

## 后续修改方法

### 方法A：网页直接修改（最简单）

1. 打开 GitHub 仓库页面
2. 点击 `ai-quiz-app.html` 文件
3. 点击右上角的 **铅笔图标**（Edit this file）
4. 修改代码
5. 点击 **Commit changes...**
6. 等待 1-2 分钟，网站自动更新

### 方法B：本地修改后上传

1. 在本地用记事本/VS Code 修改 `ai-quiz-app.html`
2. 回到 GitHub 仓库 → **Add file** → **Upload files**
3. 拖入修改后的文件，勾选覆盖原文件
4. 点击 **Commit changes**

### 方法C：Git 命令修改（推荐）

```bash
cd "文件所在目录"

# 修改文件后执行
git add ai-quiz-app.html
git commit -m "更新UI颜色"
git push
```

## 常见问题

**Q：GitHub Pages 访问慢怎么办？**
A：可以绑定自定义域名（如阿里云/腾讯云购买的域名），或者使用 jsDelivr CDN 加速。

**Q：数据会丢失吗？**
A：不会。所有数据保存在浏览器本地（localStorage），换设备需要重新打开网站。

**Q：手机能访问吗？**
A：可以。GitHub Pages 生成的网址在手机浏览器中打开即可，界面已做移动端适配。

**Q：可以分享给同学吗？**
A：可以。直接把 `https://你的用户名.github.io/ai-quiz-app/` 这个链接发给别人就行。
